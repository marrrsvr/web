Estandares de diseño web(W3C)

Seguro te ha pasado que al navegar en un sitio web, no puedes interactuar con una seccion o un objeto multimedia, como un video o imagen, aunque lo intentes con diferentes navegadores. Estas interrupciones en la navegacion son la principal razon por la que los usuarios no regresan a los sitios web, ya que piensan que no funcionan. Para resolver estas situaciones se han creado y aplicado estandares para el diseño web.

La real academia española define estandar como un tipo, modelo, norma, patron o referencia, que en terminos de diseño web significa que al creer las paginas se deben seguir ciertos criterios y normas, asi como cubrir determinados requisitos para que los elementos del hardware y del software sean compatibles entre si.

los estandares de diseño web son un conjunto de directrices y normas que garantizan la accesibilidad, compatibilidad, usabilidad y eficiencia de los sitio web. estos estandares son definidos por organizaciones como el world wide consortium(W3C), y se actualizan constantemente para adaptarse a nuevas tecnologias; aqui un resumen en la tabla 1.2.

tabla 1.2 Estandares establecidos por el W3C.
Estandar Descripcion
diseño y aplicaciones web incluye estandares para la construccion y representacion de paginas web, asi como informacion para el desarrollo de sitios con diseño inclusivo.
arquitectura web se centra en las bases que sostiene la web, incluye URL y HTTP, la creacion de protocolos XML y SOAP.
web semantica trabaja en tecnologias que permitan soportar una web de datos, construir vocabularios y escribir reglas para manejar esos datos.
tecnologia XML son estandares para trabajar las tecnologias de XML, incluye XQuery, XML Schema, XSLT, XSL-FO, intercambio eficiente de XML(EXI) y otros.
web de los servicios se basa en tecnologias como HTTP, XML, SOAP, entre otros, y se refiere al diseño tomando en cuenta los mensajes frecuentes en la web y en el software empresarial.
web de los dispositivos se trabaja en estandares que permitan el acceso a paginas web desde cualquier dispositivo, por ejemplo tabletas, telefonos moviles, computadoras portatiles y de escritorio.
navegadores y herramientas de autor son estandares exclusivos para el desarrollo de navegadores y motores de busqueda.
