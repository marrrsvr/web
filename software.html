introduccion al diseño web

el diseño web y el grafico son cosas diferentes, sin embargo, van de la mano en la creacion de un sitio en internet, pues, con la finalidad de que sea exitoso, se deben conocer dos elementos: primero, su funcionalidad depende del lenguaje de programacion y,segundo, su imagen o su presentacion; esto es basico para atraer y mantener a los usuarios en el sitio tanto para buscar informacion como para realizar alguna compra en linea.

cabe mencionar que en el diseño de cualquier sitio, tanto el diseñador web como el grafico, son responsables de la maquetacion, los colores, la distribucion, el tamaño y la forma de los diferentes elementos, asi como de otros aspectos de estilo y composicion.

paginas web y sitio

los conceptos de pagina y sitio web a menudo son confundidos, en cambio, no son lo mismo; por un lado, una pagina web es la unidad minima de un espacio en internet, mientras un sitio web es un lugar dedicado a un tema en particular que puede estar compuesto de varias paginas unidas por enlaces que facilitan la navegacion a traves de el.

pagina web

Es un documento digital que se visualiza de un navegador de internet.Puede contener texto, imagenes, videos, enlaces y otros elementos.Cada pagina web tiene una direccion unica (URL) que permite acceder a ella desde cualquier lugar del mundo a traves de internet; la pagina web es fundamental para cualquier persona y/o empresa que quiera estar presente a internet, ya sea para informar o interactuar con su audencia.

Sitio web

Es un conjunto de paginas web organizadas a traves de un navegador bajo un dominio. Un sitio web puede tener multiples paginas interconectadas.
