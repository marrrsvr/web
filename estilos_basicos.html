Estilos basicos

Las fuentes, tamaños y colores son los componentes basicos del estilo para diseñar un sitio web. aqui un poco de informacion

Tamaños

tabla 1.10 Atributos para modificar y definir tamaños
Atributos Descripcion
width ancho del elemento
height alto del elemento
vertical-align alineacion vertical
margin espacio que puede ser añadido entre elementos
padding relleno interior
float hace que el elemento aparezca como si flotara

Colores

tabla 1.11 Atributos para modificar y definir colores.
Atributo Descripcion
color color del elemento expresado en colores basicos, RGB o hexadecimal.
backgroud-color color de fondo del elemento.
backgroud-image usar imagen de fondo.
backgroud-repeat usar una imagen como un mosaico.
box-shadow aplicar un efecto de sombra.

Fuentes

tabla 1.12 Atributos para modificar y definir fuentes.
Atributo Descripcion
font-family para el tipo de letra.
font-size tamaño de letra.
font-weight normal o negrita.
font-style normal o cursiva.
text-decoraction subrayado, tachado.
text-tranform intercambiar entre mayusculas o minusculas.
