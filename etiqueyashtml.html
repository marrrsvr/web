Estructura basica de la web

Etiqueta html:

multiples etiquetas se escriben en pares, por lo regular una se ocupa para la apertura y otra para el cierre, y en medio debe ir el contenido que se quiere presentar; por ejemplo "etiqueta", texto, "cierre de etiqueta", de manera que, despues de abrir la etiqueta html, el texto para programar una pagina web

  • head: es la cabeza o el titulo del documento.
  • body: contiene el cuerpo del documento y la mayoria de las etiquetas.

Etiqueta head:

Esta etiqueta sirve para presentar informacion general acerca del documento, como el titulo con la etiqueta "title", asi como los enlaces script y hojas de estilo. Estas son de caracter tecnico e informativo, por lo general no se muestran en el navegador, solo si la informacion es util para el contenido; un resumen de otras etiquetas en la tabla 1.3

tabla 1.3 Etiquetas que se pueden usar en la seccion head.
etiqueta utilidad
meta puede usar el atributo name y content; por ejemplo, en las palabras clave para que los buscadores en linea encuentren la informacion.
style permite el uso de codigo CSS para mejorar la presentacion del documento.
script permite agregar codigos para ejecutar diversas acciones en la pagina, por ejemplo, un aviso, un saludo o saber cuantas visitas tiene la pagina.

etiqueta body

la etiqueta body es la parte central de la pagina web y donde se agregaran la mayoria de sus componentes. en la tabla 1.4 se muestran algunos de sus atributos que sirven para personalizar los diferentes elementos, como colores de texto, de enlaces, fondos, etc.

tabla 1.4 atributos de la etiqueta body
atributo funcion
bgcolor establece color de fondo
background establece una imagen de fondo
link determina el color de los enlaces no visitados
text define el color del texto
