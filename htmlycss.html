HTML(hypertext markup lenguage) o lenguaje de marcas de hipertexto es un sistema de marcado que utiliza etiquetas especificas para crear y organizar la estructura de una pagina web, gracias a este se pueden insertar objetos, como imagenes, sonidos o videos, asi como aplicar formato al texto con la finalidad de que el producto sea atractivo.aqui un poco mas de informacion !toca aqui¡
