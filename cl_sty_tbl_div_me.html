Maquetacion Web

La maquetacion web consiste en organizar los elementos de una pagina escrita en HTML para que sea igual al bosquejo diseñado en papel, a este proceso tambien se le conoce como diagramacion.

Clases

Antes de comenzar con el tema de diseño, es importante conocer que son las clases y los selectores de las hojas de estilo(CSS), que le dan color o formato a la pagina web.

En HTML y CSS, una clase(class) es un atributo que se usa para agrupar elementos y aplicarles estilos o comportamientos especificos.

Primero se indica la etiqueta o el nombre asignado al que se le aplicara el estilo y despues se configurara encerrando entre llaves:({}). El o los atributos son separados por dos puntos de su valor establecido. Algunos de estos atributos de la etiqueta style, se mostraran en la tabla 1.7 las clases se pueden usar con style y nombre de la clase con un punto al principio con por ejemplo: .texto.

tabla 1.7 Atributos de la etiqueta style.
Atributo Descripcion
Style type="text/css", (con el principio y cierre de la etiqueta). Inicio o entrada de la etiqueta.
Html{background-color:gold;} El atributo background-color sirve para establecer un color de fondo.
h1 {color:#ff0000;font-family:verdana;text-align:center;} a la etiqueta h1 se le cambia el color, el tipo de fuente y la alineacion a centar.
p {color:#ff0000;font-family:Courier New;font-size:12px;} para la etiqueta p se declara un estilo de fuente, tamaño numero 12 y un color diferente.
style(cierre de la etiqueta) fin o termino de la etiqueta.

Tablas

Para crear una tabla en html se usan las etiquetas que se describen en la tabla 1.8.

tabla 1.8 Etiquetas para crear tablas en lenguaje HTML.
Etiqueta Descripcion
table indica la creacion de una tabla y es el contenedor principal.
tr define la creacion de una fila o linea horizontal.
th define encabezados para las columnas y hace algunas variaciones en el formato del texto.
td modifica las celdas que contiene la tabla.

Divs

Esta etiqueta sirve para dividir la pagina web y se puede aplicar a un grupo especifico de parrafos, en fondos, en tipos de letra y en otras opciones de formato para dar la apariencia de que existe una division de secciones; en la siguiente tabla 1.9 se mostrara algunos atributos.

tabla 1.9 Atributos de la etiqueta div
Atributo Descripcion
id Define como unica a la etiqueta y hace referencia a ella.
style Define estilos CSS.
title Muestra un titulo del elemento.
height Permite modificar la altura, aunque se recomienda hacerlo con CSS.
width Permite modificar el ancho, aunque se recomienda hacerlo con CSS.
