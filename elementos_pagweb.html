elementos consejo
nombre es el primer elemento de busqueda en la red, debe tener relacion con el contenido o ser facil de recordar.
contenido la pagina web no solo debe ser bonita, requiere contenido actualizado e interesante para mantener a antiguos visitantes y atraer a nuevos usuarios.
imagenes es importante coloar fotografias e imagenes con buena calidad, ya que dan mala impresion aquellas que estan pixeladas o desenfocadas.
tipografias usar diferentes fuentes tipografias a las comunes, como arial o times new roman, ayuda siempre que sean facil de leer y permitan resaltar las diferentes secciones.
botones e iconos se pueden crear botones en lugar de dejar solo enlaces para hacer mas atractiva la paginas web, de modo que el usuario sienta que interactua con ella.
navegacion facil un recorrido intuitivo por la pagina otorga un experencia satisfactoria.
fondos se sugieren no usar fondos en colores fuertes porque pueden interferir con la lectura, aunque si se recomiendan para apoyar paginas con textos extensos.
enlaces a redes sociales colocar enlaces a las diferentes redes sociales para interactuar con otros usuarios o los creadores, tambien es posibles crear una comunidad virtual.
datos de contacto tener interaccion con los visitantes permite crear lealtades hacia la pagina.
